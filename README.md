# Morse Trainer Pro

Aplicacion web sin dependencias para practicar recepcion auditiva de codigo Morse desde movil o escritorio.

## Funciones

- Entrenamiento por letras, numeros o ambos.
- Respuesta escrita con teclado, sin opciones visibles.
- Partidas por tiempo: 1, 2 o 5 minutos.
- Partidas por cantidad fija de caracteres.
- Control de velocidad entre 8 y 30 WPM.
- Tono de practica a 750 Hz.
- Boton para repetir el sonido sin mostrar puntos ni rayas.
- Resumen final con aciertos, fallos, efectividad y aciertos por minuto.

## Uso

Abre `index.html` directamente en el navegador. Tambien puedes iniciar un servidor local:

```bash
python3 -m http.server 8000
```

Luego visita `http://localhost:8000`.
